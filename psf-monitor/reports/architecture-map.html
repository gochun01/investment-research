계층 구조 (5-Layer Stack)

L5 자율 계층 (Autonomy)
감지 → 판단 → 실행 → 검증 → 기억갱신 루프. 울타리 내 무승인 실행. Level 3.
SKILL-autonomy.md GUARDRAILS.md monitoring.md autonomy.py
↕ 자율 루프가 진화를 트리거
L4 진화 계층 (Evolution)
Self-Audit Q1~Q5 → 오류 적재 → F-테이블 누적 → W/Q Loop 리뷰 → 지침 수정. 시간이 지날수록 개선.
SKILL-fail-evolve.md errors.md audits/*.md
↕ 진화가 실행 품질을 높임
L3 실행 계층 (Execution)
Phase 0 → 데이터 수집 → axis-map 6단계 → PSF 판정 → 출력. D/W/M/Q 4중 루프.
SKILL-core.md CLAUDE.md ontology.md axis-summary.md axis-map.md collection.md projection-engine.md SCHEMAS.md validate.py snapshot.py
↕ 실행이 기억을 갱신
L2 기억 계층 (Memory)
단기(state/projection) + 중기(history/reports) + 장기(미구현). 세션 간 브릿지.
state.json projection.json history/*.json reports/*.html
↕ 기억이 인프라 위에 존재
L1 인프라 계층 (Infrastructure)
Python 3개 + MCP 연결 + 파일시스템. 코드 최소. 스케줄러 없음.
core/validate.py core/snapshot.py core/autonomy.py MCP 8개+ 서버
