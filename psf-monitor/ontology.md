> - 다른 운용 문서(collection.md, CE, TE)와 이 문서가 충돌하면 → 이 문서가 우선.

## 판 (Plate) — 비가역×다중관통×자기강화

### 입장 기준 (원본 §4-1)

```
3조건 동시 충족 시 판에 등록. 하나라도 빠지면 판이 아니다.

① 비가역: 되돌릴 수 있는 의사결정자가 없다
② 다중관통: 정치·경제·기술·사회 중 3개+ 동시 관통
③ 자기강화: 스스로 커지는 피드백 루프 존재

3/3 → 판 등록   |   2/3 → 판 후보 (F에 배치)   |   1/3 이하 → 판 아님
```

## 국면 판정 [고정]

```
🟢 정상: 모든 Link 비활성 + 층간 일치
🟡 경계: L7 활성 또는 Divergence 1개+
🔴 위기: L8 활성 또는 CorrFlip + L7 동시 활성

+ L3.5/CorrFlip 보조 태그 추가
```

| 전환 | 행동 |
|---|---|
| 🟢→🟡 | 포지션 점검, 헤지 검토, 모니터링 빈도 ↑ |
| 🟡→🔴 | 리스크 축소, 현금 비중 ↑, 안전자산 확인 |
| 🔴→🟡 | 저점 매수 기회 스캔, 구조 회복 신호 확인 |
| 🟡→🟢 | 정상 포지션 복귀, 모니터링 빈도 ↓ |
