# 반응 패턴 판독 사전

> Phase 4 판독 시 참조. 판독 = 구조를 읽는 것. 투자 판단 아님.

## 렌즈 1: 방향 일치도

| 패턴 | 조건 | 의미 |
|------|------|------|
| 수렴 | 4개+ 계층 같은 방향 | 컨센서스. 반전 시 충격 큼 |
| 분열 | 방향 혼재 | 소화 중. 게이트 변수 대기 |
| 괴리 | 가격과 서사가 반대 | 가장 중요한 신호 |
| 침묵 | 다수 무반응 | 무관 판단. 또는 미인지 |

괴리 세부: 가격↑서사↓ = 가격 선행. 가격↓서사↑ = 미반영 또는 과장. 가격→서사↑↓ = 노이즈.

## 렌즈 2: 시간 구조

| 유형 | 순서 | 의미 |
|------|------|------|
| A 시장주도 | 가격→서사→전문가 | 정보 비대칭 가능 |
| B 미디어주도 | 서사→가격→포지셔닝 | 재귀 루프 가능 |
| C 정책주도 | 정책→서사→가격 | 관리된 이벤트. 신뢰 여부가 핵심 |
| D 분석주도 | 전문가→서사→가격 | 초기. 확산 여부가 분기점 |
| 동시 | 모든 계층 동시 | 예정된 이벤트. 서프라이즈 없음 |

## 렌즈 3: 크기 비례성

| 판정 | 조건 | 의미 |
|------|------|------|
| 과잉반응 | 반응 > 이벤트 | 쌓인 불안 폭발. 되돌림 가능 |
| 비례반응 | 반응 ≈ 이벤트 | 합리적 소화 |
| 과소반응 | 반응 < 이벤트 | 미인지/선반영. 지연 반영 가능 |
| 무반응 | 반응 ≈ 0 | 완전 선반영 또는 무관 |

보조: ±1% 이내 = 무반응~과소. ±3~5% = 유의미. ±5%+ = 과잉 후보.

## 렌즈 4: 전파 경로

| 경로 | 후속 관찰 |
|------|----------|
| A 시장→서사 | 서사가 정당화하면 지속, 부정하면 되돌림 |
| B 서사→가격 | 포지셔닝 따라오면 구조적, 안 따라오면 노이즈 |
| C 정책→가격 | 가격이 정책 방향이면 신뢰, 반대면 불신 |
| D 전문가→서사 | Tier 1~2 채택 시 확산, 미채택 시 소멸 |

## 빈출 조합 패턴

| 조합 | 패턴명 |
|------|--------|
| 전 계층 ↑ | 완전 수렴 — 컨센서스 |
| 가격↑ 서사↓ 전문가 분열 | 가격-서사 괴리 |
| 가격→ 서사↑↓ | 노이즈 — 시장 무시 |
| 전 계층 ↓ 정책 대응 | 위기 수렴 |
| 가격↑ 서사↑ 전문가↓ 포지셔닝↓ | 스마트 괴리 — 경고 |
