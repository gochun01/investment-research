## 2. 금융 특화 규칙 부재 — 솔직한 목록

### 2-1. 밸류에이션 모델 검증

현재 equity_research에 DCF 할인율 노후화(lr_004), PER 고평가(lr_002) 정도만 있음.

| 부재 규칙 | 필요 이유 | 우선순위 |
|---|---|---|
| DCF 터미널 성장률 적정성 | 터미널 성장률 > GDP 성장률이면 비현실적. 이것만으로 DCF가 2배 차이남 | 🔴 높음 |
| WACC 구성요소 적정성 | 베타·리스크프리미엄·부채비용이 시장 현실과 괴리되는 경우 | 🟡 중간 |
| PER 밴드 / EV-EBITDA 비교 | 동종업계 평균 대비 밸류에이션 괴리 탐지 | 🟡 중간 |
| Sum-of-Parts 정합성 | 부문별 합산과 전체 밸류에이션 불일치 | 🟡 중간 |
| 목표가 민감도 분석 유무 | 핵심 가정 ±10% 변동 시 목표가 변화 범위 미제시 | 🟡 중간 |

### 2-2. 채권/크레딧 분석

현재 **완전 부재**. 회사채·국채·크레딧 분석 리포트가 들어오면 대부분 ⚫ 처리됨.

| 부재 규칙 | 필요 이유 | 우선순위 |
|---|---|---|
| 신용등급 변동 리스크 | 등급 하향 시 스프레드 급등 → 채권 가격 급락 | 🔴 높음 |
| 코베넌트(Covenant) 분석 | 재무 코베넌트 위반 시 조기상환 트리거 | 🔴 높음 |
| 이자보상배율(ICR) | 영업이익/이자비용 < 1이면 이자도 못 갚는 상태 | 🟡 중간 |
| 듀레이션/컨벡서티 적정성 | 금리 민감도 분석 | 🟡 중간 |
| 회사채 스프레드 추이 | 스프레드 확대 추세에서의 매수 추천 위험 | 🟡 중간 |

### 2-3. 한국 시장 특화

| 부재 항목 | 필요 이유 | 우선순위 |
|---|---|---|
| K-IFRS vs US-GAAP 비교 주의 | 영업이익 정의 차이, 연결 범위 차이로 수치 비교 오류 빈발 | 🔴 높음 |
| 원/달러 환율 영향 | 수출 기업 실적에 환율이 5~10%p 영향. 환율 가정 미명시 시 검증 불가 | 🔴 높음 |
| 한국 주요 섹터 Omission (8개) | 2차전지, 방산, 조선, 자동차, 은행/보험, 리츠, 유통/소비재, 통신 | 🟡 중간 |
| 공매도 규제 영향 | 한국 시장 특유의 공매도 금지/재개 이벤트 | 🟢 낮음 |
| 대주주 지분율/지배구조 | 한국 특유의 지배구조 리스크 (순환출자, 일감 몰아주기) | 🟡 중간 |

### 2-4. 펀드/ETF 분석

현재 fund_factsheet doc_type은 **사실상 무력**.

| 부재 항목 | 필요 이유 | 우선순위 |
|---|---|---|
| 벤치마크 대비 성과 비교 | 펀드가 벤치마크를 이겼는지 검증 | 🟡 중간 |
| 보수 체계 적정성 | 총보수율(TER)이 동종 대비 과도한지 | 🟡 중간 |
| 집중도 리스크 | 상위 10개 종목 비중, 섹터 편중 | 🟡 중간 |
| 유동성 리스크 | AUM 대비 일 거래량, 환매 정책 | 🟡 중간 |

### 2-5. 파생상품/구조화상품

현재 **완전 미지원**. doc_type 자체가 없음.

| 부재 항목 | 필요 이유 | 우선순위 |
|---|---|---|
| 구조화상품 페이오프 분석 | 조건부 수익 구조(KI/KO) 검증 | 🟢 낮음 (특수) |
| 옵션 그릭스 적정성 | 델타/감마/베가 산출 근거 | 🟢 낮음 (특수) |
| 기초자산 상관관계 | 다중 자산 연계 상품의 상관관계 가정 | 🟢 낮음 (특수) |

|---|---|---|

| 원/달러 환율 가정 규칙 (lr_023) 추가 | 30분 | 수출 기업 분석 시 환율 전제 검증 |

| 작업 | 예상 시간 | 효과 |
|---|---|---|

### Phase C: 중기 (3~6개월, 오답노트 기반)

| 작업 | 전제 조건 | 효과 |
