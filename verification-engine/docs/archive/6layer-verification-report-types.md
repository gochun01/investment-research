```
│   → "왜 믿을 수 있는가"를 보여줌

└── 원칙 3: 액션 가능성 (Actionable)
```

# 유형 B — 레이어드 인텔리전스 리포트 (Layered Intelligence Report)

> "드릴다운 가능한 기관급 HTML 리포트"

---

## B-1. 개요

```
컨셉: 전문 투자자가 5~10분에 전체 구조를 파악하고
      필요 시 소스까지 추적 가능
포맷: 인터랙티브 HTML (접기/펼치기 섹션)
영감: Morgan Stanley Blue Paper, Bridgewater Daily Observations
      + Hebbia AI의 소스 링크 투명성
      + EY·KPMG 투명성 리포트의 방법론 공개 원칙
용도: 유료 구독 프리미엄 / 기관 투자자 납품
```

---

## B-2. 3단계 드릴다운 구조

```
Level 1 — Executive Layer (기본 보기)
│  종합 판정 + 신뢰도 바
│  최고 리스크 플래그 1개
│  KC 트리거 여부
│  유효기간 + 액션 권고
│  [▶ 상세 보기] 버튼
│
Level 2 — Evidence Layer (클릭 시 펼침)
│  6층 판정 테이블 (층별 판정 + 근거 1줄)
│  Claim별 결과 테이블
│  → 각 Claim 옆에 MCP 소스 직접 링크
│    예: "DART에서 확인된 수치: PER 12.3x [링크]"
│  BBJ Break 목록 (반론 + 문서 언급 여부)
│  [▶ 원본 데이터 보기] 버튼
│
Level 3 — Raw Evidence Layer (전문가 전용)
   MCP 조회 원본 데이터
   → FRED 시리즈 실제값 + 조회 시점
   → DART 공시 원문 링크
   KC 추출 논리 전문
   RULES.md 적용 규칙 ID 전체 목록
   JSON 다운로드 버튼 (verification_store v1.1)
```

# 유형 C — PSF 검증 브리핑 (PSF Verification Brief)

> "판-구조-흐름 프레임워크와 검증 결과를 통합한 독창적 포맷"

---

## C-1. 개요

```
컨셉: 6-Layer 검증 결과를 PSF 동심원 모델에 통합
      리포트의 주장이 현재 판(레짐)과 정합하는가를 추가 판정
포맷: 구조화 HTML or 마크다운
영감: 없음 — 완전한 독창 포맷
      글로벌 어디에도 없는 PSF × 검증 결합
용도: 유료 구독 핵심 상품 / PSF 시스템 사용자 전용
차별화: 이것이 희발님 시스템의 실제 차별화 원점
```

---

## C-2. PSF 검증 브리핑의 핵심 아이디어

```
기존 검증 엔진 (6-Layer)
└── "리포트의 수치·논리·생략이 맞는가?" 판정

PSF 검증 브리핑 추가 레이어
└── "리포트의 결론이 현재 판(레짐)과 정합하는가?" 판정
    → 수치가 맞아도 판이 틀리면 결론이 무효
    → 동심원 모델: 바깥원(판) 변경 시 안쪽(결론) 자동 무효
```

## C-5. 6층 검증 vs PSF 정합성 — 2×2 매트릭스

```
          PSF 정합성
          🟢 일치    🔴 불일치

        │ 신뢰   │ 구간   │
     🔴 ├────────┼────────┤
        │ 전제   │ 신뢰   │
        │ 검토   │ 불가   │
        └────────┴────────┘

→ 6층 🟢 + PSF 🔴: 수치는 맞지만 판이 틀림 → 주의 구간
→ 6층 🔴 + PSF 🟢: 판은 맞지만 리포트 품질 낮음 → 전제 검토
→ 6층 🔴 + PSF 🔴: 이중 실패 → 신뢰 불가
```
