# 글로벌 매크로 프레임워크 — 8축 환경 읽기 v1.0

## 이 문서의 역할
매크로 분석 시스템의 뼈대. 8축 정의, 4-Layer 지표 계층, 인과 체인, 레짐 판정 규칙을 담는다.

---

## 핵심 질문

"지금 돈이 위험자산 방향으로 흐르고 있는가"

이 질문에 7개 필수 변수가 답한다:

```
A1 Core PCE     → 수도꼭지를 틀 사람(Fed)이 틀 준비가 됐는가
A2 중국 Credit  → 미국 밖에서도 물이 나오고 있는가
B1 실질금리     → 나온 물이 국채에 멈추는가, 더 가는가
B2 DXY          → 물이 미국 안에 갇히는가, 밖으로 퍼지는가
B3 USD/JPY      → 캐리트레이드라는 숨은 배수구가 열리려 하는가
B4 Net Liquidity → 실제로 물이 얼마나 나와 있는가 (양)
B5 HY 스프레드  → 배관이 막히기 시작했는가 (시스템 리스크)
```

한 문장으로:
**"물이 나오고 있고(A1+A2+B4), 막히지 않았고(B5), 갇히지 않았고(B1+B2), 역류하지 않는다면(B3) → 위험자산에 돈이 간다."**

## 8축 정의

### 축 1: 유동성 양 — "얼마나 나왔는가"
핵심: Net Liquidity (WALCL−TGA−RRP), 글로벌 M2
해석: 양이 늘면 자산 가격 상승 압력. 줄면 하락 압력.
지표: B4 Net Liq, C4 글로벌 M2, D3 RRP, D4 TGA, D5 WALCL

### 축 2: 유동성 가격 — "돈의 비용이 어떤가"
핵심: 실질금리 (DFII10), 수익률 곡선 (T10Y2Y)
해석: 비용이 높으면(실질금리 +) 국채가 매력적 → 위험자산 역풍. 낮으면 반대.
지표: B1 실질금리, C3 T10Y2Y, C10 DFF, D10 FedWatch

### 축 3: 달러/외환 — "돈이 어디로 가는가"
핵심: DXY, USD/JPY, USD/CNY
해석: 달러 강세 = 미국 안에 갇힘 + EM 역풍. 약세 = 분산 + 대안자산 순풍.
엔캐리: USD/JPY 급변은 정상 경로 무시하고 글로벌 매도 유발 (Override).
지표: B2 DXY, B3 USD/JPY, C9 USD/CNY, D9 CFTC JPY

### 축 4: 경기 사이클 — "실물 경제가 어디에 있는가"
핵심: Core PCE (키스톤), ISM PMI, 실업률
해석: Core PCE가 Fed 행동을 결정. PMI가 확장/수축. 실업률이 비상 트리거.
지표: A1 Core PCE, C5 ISM PMI, C6 실업률

### 축 5: 신용/금융 안정 — "배관이 막혔는가"
핵심: HY 스프레드, SOFR, 은행 지급준비금
해석: 스프레드 확대 = 위험 기피. SOFR 급등 = 단기 자금 경색. L7 핵심 입력.
지표: B5 HY OAS, D6 SOFR, D2 지급준비금, D1 SLOOS

### 축 6: 리스크 환경 — "공포가 지배하는가"
핵심: VIX, MOVE Index
해석: VIX 30+ = 모든 상관관계가 1로 수렴 (동반 매도). Override 조건.
MOVE = 채권시장 변동성. VIX와 동반 급등 시 L7/L8 접근.
지표: C1 VIX, C2 MOVE

### 축 7: 지정학/에너지 — "외부 충격이 있는가"
핵심: Brent 유가, BEI (기대 인플레이션)
해석: 유가 급등 → BEI 상승 → Core PCE 압력 → Fed 인하 차단.
이 경로가 정상 경로(Fed 경로)를 역전시킬 수 있음.
지표: C7 Brent, C8 BEI

### 축 8: 자본 흐름 — "구조적 힘이 어디를 가리키는가"
핵심: 중국 Credit Impulse, 재정적자/GDP, 텀 프리미엄
해석: 중국 완화 → 글로벌 M2 확장. 재정적자 확대 → 국채 공급 → 텀 프리미엄 상승 → 실질금리 상승 (Fed 인하와 상충).
지표: A2 중국 Credit, D7 재정적자/GDP, D8 텀 프리미엄

## 4-Layer 지표 계층

### Layer A — 근본 (2개)
움직이면 나머지가 따라온다. 매주 반드시 확인.

| ID | 지표 | 소스 | 왜 근본인가 |
|----|------|------|-----------|
| A1 | Core PCE YoY | FRED PCEPILFE | 키스톤. Fed의 유일한 바인딩 제약. 이것이 2.0%에 도달해야 Fed가 인하 재개 |
| A2 | 중국 Credit Impulse | PBOC 월간 대출 | 글로벌 M2의 30%. 미국 밖 유동성의 최대 변수 |

### Layer B — 전달 (5개)
Layer A의 결과이자 시장의 직접 원인. 매주 반드시 확인.

| ID | 지표 | 소스 | 역할 |
|----|------|------|------|
| B1 | 실질금리 | FRED DFII10 | 자금 배분 방향. 양(+) = 국채 흡인, 음(−) = 위험자산 순풍 |
| B2 | DXY | Yahoo DX-Y.NYB | 달러 방향 = 글로벌 자본 흐름 방향 |

| B4 | Net Liquidity | WALCL−TGA−RRP | Fed 수조에서 실제 방류된 양 |
| B5 | HY OAS | FRED BAMLH0A0HYM2 | 신용 환경 = L7 핵심 입력 |

### Layer C — 확인 (10개)
Layer B의 방향을 교차 확인. 일치하면 "확인", 불일치 시 경고.

## 인과 체인 (텍스트 지식 그래프)

### 정상 경로 2개

```
경로 1 — Fed 경로 (가장 지배적):
A1 Core PCE ↓
  → Fed 인하 결정
    → B1 실질금리 ↓
      → 국채 매력 감소 → 자금이 위험자산으로 이동
    → B4 Net Liquidity ↑
      → 시중 유동성 증가
    → B2 DXY ↓
      → 달러 약세 → 글로벌 자본 분산
    → B5 HY 축소
      → 신용 환경 완화 → 위험자산 선호
  → 최종: 위험자산 순풍

경로 2 — 중국 경로:
A2 중국 Credit Impulse ↑
  → C4 글로벌 M2 ↑
    → B2 DXY ↓ (달러 상대 약세)
      → EM + 대안자산 유입
  → 최종: 위험자산 순풍 (Fed 경로와 합류 시 가속)
```

### 교란 경로 3개 (정상 경로를 무시하거나 역전)

```
교란 1 — 엔캐리 경로 (Override):
B3 USD/JPY 급변 (엔 강세)
  → 캐리트레이드 청산 → 글로벌 위험자산 동반 매도
  → B5 HY 확대 → L7 접근
  → A1/A2/B1/B2와 무관하게 작동
  → 사례: 2024년 8월 글로벌 급락

교란 2 — 유가/지정학 경로 (역전):
C7 Brent 급등 (호르무즈 봉쇄, 전쟁 등)
  → C8 BEI 상승 → A1 Core PCE 상승 압력
  → Fed 인하 차단 → B1 실질금리 상승
  → Fed 경로를 역전시킴
  → 사례: 이란 위기 시나리오

교란 3 — 재정 경로 (상충):
D7 재정적자 확대 → 국채 대량 발행
  → D8 텀 프리미엄 상승
  → B1 실질금리 상승 (Fed가 인하해도)
  → 통화정책과 재정정책의 상충 (재정 지배)
  → 사례: 2023년 가을 국채 금리 급등
```

### 경로 상태 판별

```
매주 판별할 것:
├── 지배 경로: 이번 주 어떤 경로가 시장을 움직이는가
├── 활성 교란: 교란 경로가 작동 중인가 (있으면 어떤 것)
├── 상충: 경로 간 상충이 있는가
│   → 예: DXY↓(Fed 경로) vs B1↑(재정 경로) = 상충 → Transition
└── 전환: 전주 대비 경로가 바뀌었으면 왜
```

## 레짐 판정

### L7/L8 최우선 (시스템 리스크 게이트)

```
L7 = 0.30×HY정규화 + 0.25×VIX정규화 + 0.20×SOFR갭 + 0.15×MOVE정규화 + 0.10×TED
L8 = 0.35×은행CDS + 0.30×TED + 0.20×유동성동결 + 0.15×긴급FOMC확률

정규화: (현재값−정상하한) / (위기상한−정상하한), 클리핑 [0,1]
  HY: (현재−250) / (800−250)
  VIX: (현재−12) / (45−12)
  MOVE: (현재−80) / (200−80)
  SOFR갭: (현재−0) / (50bp−0)

Threshold: 0.60

판정:
  🟢 정상 (L7<0.60, L8<0.60): 레짐 판정 진행
  🟡 경계 (L7≥0.60, L8<0.60): 위험자산 축소, L8 집중
  🔴 위기 (L7≥0.60, L8≥0.60): 현금 극대화, 정책 대응 후 재진입
```

### 레짐 판정 (L7/L8 정상일 때)

```
Layer B 5개의 방향으로 판정:

각 지표의 "위험자산 방향" 정의:
  B1 실질금리: 하락 추세 = 위험자산 방향
  B2 DXY: 하락 추세 = 위험자산 방향
  B3 USD/JPY: 안정 (급변 없음) = 위험자산 방향
  B4 Net Liquidity: 상승 추세 = 위험자산 방향
  B5 HY 스프레드: 축소 추세 = 위험자산 방향

레짐:
  🟢 팽창기: 5개 중 4개+ 위험자산 방향
  🟡 Transition: 2~3개 전환, 방향 미확정
  🔴 수축기: 4개+ 안전자산 방향
  🟡 과열기: 팽창 + 과열 신호 (VIX<12, 스프레드 역사적 저점)

전환 규칙:
  ├── 레짐 변경 시 반드시 어떤 지표가 방향을 바꿨는지 명시
  ├── 1개 지표 전환 = 아직 레짐 변경 아님 (관찰)
  └── 2개+ 동시 전환 = 레짐 전환 검토
```

### 쿼드런트 보조 판정 (v3.0 — TE에서 이관)

> 기존 레짐 판정(L7/L8 + Layer B 방향 수)에 **경기-인플레 4쿼드런트**를 보조로 추가.
> TE ① 레짐 판정기에서 이관. 레짐 판정의 "해석 보강"으로 사용.
> 기존 레짐 판정을 대체하지 않는다. 병기한다.

```
4쿼드런트 매트릭스:

              인플레 ↑              인플레 ↓
  성장 ↑     Q2 Reflation          Q1 Goldilocks
  성장 ↓     Q3 Stagflation        Q4 Deflationary

판정 프록시:
  성장축: C5 ISM PMI (50 기준)
    PMI > 50 → 성장 ↑
    PMI ≤ 50 → 성장 ↓
    PMI 미발표 시: C6 실업률 보조 (4.5% 기준)

  인플레축: A1 Core PCE (2.5% 기준)
    Core PCE > 2.5% → 인플레 ↑
    Core PCE ≤ 2.5% → 인플레 ↓
    보조: C8 BEI (T5YIE) 방향

출력 형식:
  레짐 판정 옆에 병기: "🟡 Transition (Q2 Reflation)"
  보고서 §2에 쿼드런트 표시 포함.

전환 감지:
  쿼드런트 변경 시 보고서에 "쿼드런트 전환: Q2→Q3 (사유)" 명시.
  Layer B 레짐과 쿼드런트가 상충 시 → 보고서에 주석 ("Layer B 팽창 but Q3 스태그 진입 접근").
```

## 키스톤 구조

```
현재 환경에서의 인과 지배 구조:

Core PCE (A1)
  → Fed 행동을 결정
    → 9개 지표가 연쇄 반응 (B1~B5, C3, C10, D5, D10)
    → 이 9개가 8축 중 5축을 커버

따라서: Core PCE 월간 발표가 매크로 판정의 가장 중요한 단일 이벤트

키스톤 전환 조건 (Core PCE가 키스톤이 아니게 되는 경우):
  ├── 실업률 5%+ 급등 → 바인딩 제약이 고용으로 전환
  ├── 금융 위기 (L7/L8 활성) → 키스톤 소멸, 긴급 대응
  └── 스태그플레이션 → 두 제약 동시 활성, Fed 딜레마

보고서에서: "현재 키스톤 상태" 명시
  ├── Core PCE 바인딩: 정상 (현재)
  ├── 실업률 바인딩: 고용 악화 시
  └── 위기 모드: L7/L8 활성 시
```

## 이벤트 연결 서사 메커니즘

```
Step 1: Layer B 5개 중 전주 대비 유의미 변화 식별
  기준: ±5% 또는 절대 임계 (B1: ±20bp, B3: ±3%, B5: ±50bp)

Step 2: 변화 원인 = 이벤트 탐색 (웹검색)
  "이번 주 글로벌 금융시장 핵심 이벤트"

Step 3: 이벤트 → 지표 인과 경로 매핑
  "호르무즈 봉쇄 → C7 Brent $95 → C8 BEI +20bp → A1 Core PCE 압력 → Fed 인하 차단"

Step 4: 경로가 레짐에 미치는 영향
  "교란 경로 2 (유가) 활성화. Fed 경로 역전. Transition 유지."

Step 5: 3~5문장 서사 압축 → 1페이지 요약 삽입

이벤트 없는 주:
  "주요 이벤트 없음. Fed 경로 지배 유지.
   B1 실질금리 +1.75% (전주 +1.80%), 점진 하락 추세.
   Transition 유지, 전환 트리거 대기."
```

## 내러티브 상태 추적

```
"Good news is good news" 레짐:
  강한 경제 데이터(고용↑, PMI↑) → 주가 상승
  = 성장 기대 > 긴축 우려
  = 위험자산 우호

"Good news is bad news" 레짐:
  강한 경제 데이터 → 주가 하락
  = 긴축 우려 > 성장 기대
  = 위험자산 역풍

판별 방법:
  고용 보고서/ISM 발표 직후 S&P 500 반응으로 판별
  ├── 강한 데이터 + 주가↑ = Good is Good
  ├── 강한 데이터 + 주가↓ = Good is Bad
```
