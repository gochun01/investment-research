# 글로벌 매크로 분석 시스템 — 프로젝트 가이드

```
프로젝트명: Global Macro Environment Reader
버전:       v1.0
상태:       1단계 설계 중
작성일:     2026-03-06
환경:       Cowork + Opus 4.6 + 로컬 저장
```

---

## 1. 프로젝트 목적

**"지금 돈이 위험자산 방향으로 흐르고 있는가"를 매주 판정하는 독립 매크로 보고서 발행 시스템.**

하위 섹터(크립토, AI, 주식) 분석과 독립적으로 작동.
매크로 환경 판정 자체가 최종 결과물.

---

## 2. 핵심 질문 3개

이 시스템이 매주 답해야 하는 질문:

```
① 유동성 환경: 돈이 풀리는 판인가, 조이는 판인가
② 경기/금리 사이클: 확장/과열/수축/위기 중 어디인가
③ 리스크 환경: 시스템이 안정적인가 (L7/L8)
```

---

## 3. 설계 원칙

```
원칙 1: 지표 27개를 4-Layer로 계층화
  → Layer A(근본 2) + B(전달 5) = 7개 필수
  → 나머지 20개는 교차 확인/배경

원칙 2: 온톨로지는 텍스트 인과 체인
  → DB 그래프 아님. 보고서에 "이번 주 스토리"로 서술
  → DB는 보고서가 안정된 후 Phase 4에서

원칙 3: 보고서가 최종 결과물
  → 대시보드는 보조 도구
  → 핵심은 인과 체인 서사 + 레짐 판정 + 행동 함의

원칙 4: 이벤트 연결 서사
  → 매주 지표 변화의 원인(이벤트)을 식별하고
  → 이벤트 → 지표 → 레짐 영향의 인과 체인을 서술

원칙 5: DB 없이 로컬 파일
  → invest_ontology 연결은 Phase 4
  → 지금은 MD 보고서 + 로컬 저장 + Notion 아카이브
```
