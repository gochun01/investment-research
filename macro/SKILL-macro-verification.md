# 글로벌 매크로 검증 프레임워크

## 이 문서의 역할
매크로 보고서의 지표·추론·전제를 3계층으로 검증한다.
범용 검증 표준 뼈대를 매크로 환경 분석에 특화.

### 괴리 기준
```
숫자 지표: 보고서 값 vs MCP 최신값 ±2% 이상 → ⚠️ 플래그
방향 지표: 보고서 방향 vs 실제 방향 불일치 → ⚠️ 플래그
괴리 시: 재수집 → 수정 → 수정 사유 명시
```

## Layer 2: 추론 검증

### 매크로 특화 추론 오류 패턴

| 오류 유형 | 매크로에서의 예시 | 점검 방법 |
|---------|-----------------|---------|
| 단일 변수 과의존 | "Core PCE가 전부" | Layer B 5개 중 Core PCE 무관 변수(B3 엔캐리) 별도 점검 |
| 인과 방향 오류 | "DXY↓ 때문에 위험자산↑" | 제3변수(리스크 선호) 가능성 점검 |
| 시차 무시 | "M2↑ = 즉시 주가↑" | M2→위험자산 시차 3~6개월 고려 |
| 교란 경로 무시 | Fed 경로만 보고 판정 | 엔캐리/유가/재정 교란 3개 각각 점검 |
| 내러티브 편향 | 결론을 먼저 정하고 지표 선택 | Layer C 불일치 개수로 역검증 |
| 역사적 유추 오남용 | "2008년과 같다" | 유사점 + 차이점 반드시 병기 |

### 레짐 판정 교차 검증

판정 후 반드시:
```
1. "이 판정이 틀리려면 어떤 지표가 반대로 움직여야 하는가?"
2. "Layer B와 Layer C 방향 불일치가 3개 이상인가?" → 교란 의심
3. "직전 주 판정과 달라졌다면 어떤 경로의 어떤 지표 때문인가?"
4. "이벤트 서사가 지표 변화를 충분히 설명하는가?"
```

### 인과 체인 검증

```
매주 점검:
├── 정상 경로 1(Fed): A1→B1→B4→B2→B5 방향 일관성
├── 정상 경로 2(중국): A2→C4→B2 방향 일관성
├── 교란 경로 활성 여부: B3 급변? C7 급등? D8 급등?
└── 경로 간 상충: 있으면 Transition, 없으면 레짐 확정
```

## Layer 3: 전제 검증 (월간)

### 전제 P1: Fed 반응 함수가 Core PCE에 바인딩
- 역할: 키스톤 구조의 기초
- 근거: Fed 듀얼 맨데이트, 현재 실업률 안정 → 물가만 제약
- 반론: 실업률 급등 시 바인딩 전환. 금융 위기 시 양쪽 무시.
- 검증: 매 FOMC 성명서에서 "물가" vs "고용" 언급 비중 확인
- 붕괴 시: 키스톤이 Core PCE → 실업률 또는 위기 모드로 전환
- 취약성: 🟡 중간

### 전제 P2: 엔캐리 청산이 글로벌 리스크의 숨은 변수
- 역할: B3 USD/JPY를 Layer B에 포함시킨 이유
- 근거: 2024.08 글로벌 급락, BOJ 정책 전환 시 반복 가능
- 반론: 엔캐리 규모가 과거보다 축소되었을 가능성. BOJ 소통 개선.
- 검증: CFTC JPY 순포지션 + BOJ 발언 추적
- 붕괴 시: B3를 Layer C로 격하 가능
- 취약성: 🟡 중간

### 전제 P3: 재정 지배가 통화정책을 압도
- 역할: 교란 경로 3의 기초. "Fed가 인하해도 금리가 안 내려갈 수 있음"
- 근거: 적자/GDP 6%+, 이자 비용 $1T+, 국채 대량 발행
- 반론: 달러 기축통화 특권으로 수요 유지. 텀 프리미엄이 아직 제한적.
- 검증: 텀 프리미엄(D8) 추이 + 국채 입찰 결과 (bid-to-cover)
- 붕괴 시: 교란 경로 3 비활성화, Fed 경로가 순수하게 작동
- 취약성: 🟡 중간

### 전제 P4: 중국 경기가 글로벌 수요의 30%
- 역할: A2를 Layer A에 포함시킨 이유
- 근거: 중국 GDP 글로벌 18%, 원자재 수요 30~50%, PBOC 유동성 영향
- 반론: 중국 경제의 구조적 둔화 (부동산, 인구). 영향력 감소 추세.
- 검증: 중국 PMI + 원자재 가격 + USD/CNY 추이
- 붕괴 시: A2의 중요도 하락, Layer C로 격하 검토
- 취약성: 🟢 낮음

### 전제 P5: 유가→인플레 경로가 지정학에 의해 비선형
- 역할: 교란 경로 2의 기초
- 근거: 호르무즈 봉쇄 시 Brent $90-130+, Core PCE 직접 영향
- 반론: 미국 셰일 혁명으로 에너지 자급. 유가 영향 감소.
- 검증: Brent vs BEI 상관 추적 + 지정학 이벤트 모니터링
- 붕괴 시: 교란 경로 2 비활성화
- 취약성: 🟡 중간

### 전제 취약성 요약

| 전제 | 결론 영향도 | 검증 가능성 | 취약성 | 점검 주기 |
