# 글로벌 매크로 주간 보고서 — 출력 템플릿

## 이 문서의 역할
/macro-weekly 실행 시 이 형식에 따라 보고서를 생성한다.
[ ] 대괄호 항목을 실제 내용으로 교체.

> **v2 변경**: 보고서는 반드시 "핵심 3줄 + 해석"으로 시작한다.
> 기존 템플릿은 참고 구조. 내용이 형식을 결정한다.
> 변화 없는 지표의 반복 서술 금지. 변화 있는 것만 상세.

## 핵심 3줄 + 해석 (보고서 맨 앞, 필수)

핵심 3줄:
  1줄: 레짐 + 변화 방향 — "[🟢/🟡/🔴] [점수]. [전주 대비 변화 한 줄]."
  2줄: 인과 체인 — "[이벤트] → [경로] → [결과]."
  3줄: 다음 주 감시 — "[가장 가까운 분기점]."

해석 (각 줄마다 2~3문장, 개인투자자 수준):
  1줄 해석: "..."
  2줄 해석: "..."
  3줄 해석: "..."

## 작성 가이드

### 핵심 결론 작성법
```
좋은 예:
"호르무즈 봉쇄로 Brent $95 → BEI 급등 → Fed 인하 차단 →
실질금리 +2.1% 상승. 교란 경로(유가) 활성. Transition 유지."

나쁜 예:
"이번 주 매크로 환경은 혼조세를 보였다." (이벤트 없음, 인과 없음)
```

### 이벤트 서사 작성법
```
구조: [이벤트] → [1차 영향 지표] → [2차 전달 지표] → [레짐 영향]
반드시: 이벤트가 어떤 인과 경로를 활성화했는지 명시
상충 시: 양쪽 경로 모두 서술 + "상충으로 Transition"
```

### Layer별 상세 작성법
```
Layer A+B (7개): 값 + 전주 대비 + 방향 + 왜
Layer C (10개): B와 불일치 시에만 상세. 일치 시 "B[X] 확인" 한 줄.
Layer D (10개): 변동 시에만. 없으면 "전주 유지".
```
