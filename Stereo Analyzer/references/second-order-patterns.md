# 2차 효과 패턴 라이브러리 (Second-Order Effect Patterns)

> L6 2차 효과 체인에서 참조하는 패턴 라이브러리.
> 기사가 다루지 않는 파생 효과를 구조적으로 도출하기 위한 사고 도구.

---

## 사용법

1. L4(인과 역추적)와 L5(구조 레이어)를 완료한 후 참조한다
2. 이슈의 1차 효과에서 어떤 2차 효과가 파생되는지 패턴 매칭한다
3. 해당 패턴의 메커니즘을 현재 이슈에 적용하여 구체적 2차 효과를 도출한다
4. 시간 프레임(단기/중기/장기)과 비선형 연결을 명시한다
5. 패턴이 맞지 않으면 강제 적용하지 않는다 — 패턴은 사고의 촉매이지 답이 아니다

---

## 패턴 목록

### 1. 규제 → 회피 → 혁신 (Regulation Breeds Innovation)

```
패턴명: 규제-회피-혁신 순환
메커니즘:
  규제가 강화되면 → 규제를 피하려는 우회 시도 →
  우회 과정에서 새로운 기술/비즈니스 모델 탄생 →
  새 모델이 기존 시스템보다 효율적이면 → 구조 전환 가속

역사적 사례:
  - 중국 AI칩 수출규제 → 화웨이 Ascend 자체 개발 가속 → 중국 GPU 생태계 형성
  - 은행 규제 강화 (바젤III) → 핀테크/DeFi 부상 → 전통 금융 시스템 도전
  - 환경 규제 강화 → 탄소배출권 시장 → 그린테크 산업 탄생

탐지 신호:
  - "규제 강화" + "대안 기술 투자" 동시 보도
  - 규제 대상 기업의 R&D 비용 급증
  - 규제 회피 스타트업의 투자 유치 증가

전형적 타임라인:
  단기 (0~6개월): 규제 대상 기업 타격, 준수 비용 증가
  중기 (6~18개월): 회피/대안 모델 등장, 초기 투자 집중
  장기 (2~5년): 대안 모델이 시장 점유, 원래 규제 의도와 다른 결과
```

### 2. 보조금 → 과잉 → 붕괴 (Subsidy → Overcapacity → Collapse)

```
패턴명: 보조금-과잉투자-구조조정 사이클
메커니즘:
  정부 보조금/인센티브 → 과잉 진입 (수익성 무시한 투자) →
  공급 과잉 → 가격 하락 → 한계 기업 퇴출 →
  생존 기업의 시장 지배력 강화 (승자 독식)

역사적 사례:
  - 중국 태양광 보조금 → 과잉 설비 → 가격 폭락 → 유럽 기업 퇴출 → 중국 독점
  - 한국 조선업 지원 → 과잉 수주 → 해운 불황 시 구조조정
  - EV 보조금 → 신규 진입 폭증 → 가격 전쟁 → 약자 도태 예상

탐지 신호:
  - 정부 보조금 규모 급증 + 신규 진입자 수 급증
  - 산업 전체 가동률 하락
  - "출혈 경쟁", "치킨 게임" 보도 증가

전형적 타임라인:
  단기 (0~2년): 보조금 수혜 기업 호황, 주가 상승
  중기 (2~4년): 과잉 설비 누적, 마진 압축 시작
  장기 (4~7년): 구조조정, 약자 퇴출, 생존자 독점
```

### 3. 공포 → 매도 → 저가매수 기회 (Fear → Sell-off → Opportunity)

```
패턴명: 공포-투매-역발상 사이클
메커니즘:
  공포 이벤트 발생 → 감정적 매도 (펀더멘탈 무관) →
  가격이 내재가치 이하로 하락 →
  스마트 머니 저가 매수 → 가격 회복

역사적 사례:
  - 2020.03 코로나 쇼크 → VIX 82 → S&P 33% 급락 → 이후 V자 회복
  - 2022.06 루나/셀시우스 → 크립토 공포 → BTC 15K → 이후 60K+ 회복
  - 각종 지정학 쇼크 (9/11, 러시아-우크라이나) → 단기 매도 → 중기 회복

탐지 신호:
  - VIX 30+ 또는 Fear & Greed Index 극단적 공포
  - 펀더멘탈 변화 없이 가격만 급락
  - "이번에는 다르다" 내러티브 확산 (실제로는 비슷한 패턴)

전형적 타임라인:
  단기 (0~2주): 공포 확산, 투매
  중기 (1~3개월): 가격 안정화, 스마트 머니 유입
  장기 (6~12개월): 회복 또는 새 평형 도달

주의:
  이 패턴은 "펀더멘탈이 변하지 않았을 때"만 적용.
  구조적 변화가 동반된 하락은 V자 회복하지 않을 수 있다.
  → L5 구조 판정과 교차 검증 필수.
```

### 4. 기술대체 → 일자리 → 정치반발 (Tech Displacement → Political Backlash)

```
패턴명: 기술대체-고용충격-정치반응 사이클
메커니즘:
  새 기술이 기존 일자리를 대체 → 실업/소득 감소 →
  피해 집단의 정치적 조직화 → 규제/보호 요구 →
  기술 도입 속도 조절 또는 보상 체계 도입

역사적 사례:
  - 자동화 → 제조업 일자리 감소 → 러스트벨트 정치 세력화 → 트럼프 당선
  - 우버/리프트 → 택시 산업 타격 → 각국 라이드셰어 규제
  - AI 코딩/번역 → 화이트칼라 위협 → EU AI법 등 규제 논의 가속

탐지 신호:
  - "일자리 대체" + "노조 반발" 동시 보도
  - 피해 산업 종사자 수가 선거구 수준으로 유의미
  - 정치인들의 기술 규제 발언 증가

전형적 타임라인:
  단기 (0~1년): 기술 도입 가속, 생산성 향상 보도
  중기 (1~3년): 고용 충격 가시화, 사회적 논의 시작
  장기 (3~5년): 정치적 반발 → 규제 또는 보상 체계 도입
```

### 5. 금리상승 → 좀비기업 정리 → 건전화 (Rate Rise → Zombie Cleanup)

```
패턴명: 금리-좀비정리-시장건전화
메커니즘:
  금리 인상 → 차입 비용 증가 →
  이자보상배율 1 미만 기업(좀비) 정리 →
  살아남은 기업에 자원 재배분 → 시장 건전화

역사적 사례:
  - 2022~2023 Fed 금리 인상 → 크립토/핀테크 좀비 정리 (FTX, SVB 등)
  - 2000 닷컴 버블 후 금리 인상기 → 수익 없는 닷컴 기업 대량 퇴출
  - 일본 제로금리 장기화 → 좀비기업 존속 → 생산성 저하 (역사례)

탐지 신호:
  - 금리 인상 + 기업 부도율 상승
  - 하이일드 스프레드 확대
  - "구조조정", "워크아웃" 보도 증가

전형적 타임라인:
  단기 (0~6개월): 금리 인상 충격, 시장 전반 하락
  중기 (6~18개월): 한계 기업 퇴출, 시장 정화
  장기 (2~3년): 생존 기업 수익성 개선, 시장 재평가
```

### 6. 독점 → 혁신둔화 → 신규진입 (Monopoly → Stagnation → Disruption)

```
패턴명: 독점-정체-파괴적 혁신 사이클
메커니즘:
  시장 독점/과점 → 혁신 인센티브 감소 →
  고객 불만 축적 + 기술 변화 →
  새로운 진입자가 다른 방식으로 시장 침투

역사적 사례:
  - IBM 메인프레임 독점 → PC 혁명 (애플, MS)
  - 노키아 피처폰 독점 → 스마트폰 전환 (애플 아이폰)
  - 전통 은행 독점 → 핀테크/네오뱅크 부상

탐지 신호:
  - 독점 기업의 R&D 투자 대비 혁신 성과 감소
  - 고객 불만/이탈 지표 상승
  - 인접 기술 분야에서 대안 모델 등장

전형적 타임라인:
  단기: 독점 기업 안정적 수익 (위기감 없음)
  중기 (3~5년): 대안 모델 성장, 틈새시장 공략
  장기 (5~10년): 티핑 포인트 도달, 독점 기업 지위 위협
```

### 7. 전쟁 → 공급차단 → 대체경로 가속 (War → Supply Cut → Alternative Acceleration)

```
패턴명: 충돌-공급단절-대체경로 사이클
메커니즘:
  지정학 충돌/전쟁 → 특정 공급망 차단 →
  단기 가격 급등 + 공포 →
  대체 공급원/경로 개발 가속 →
  원래 공급자의 장기적 지위 약화

역사적 사례:
  - 러시아-우크라이나 전쟁 → 유럽 가스 공급 차단 → LNG 인프라 가속 → 러시아 가스 의존도 급락
  - 1973 오일쇼크 → 중동 원유 의존 위기 → 북해 유전 개발 + 에너지 효율화
  - 미중 반도체 전쟁 → 중국 공급망 제한 → 인도/베트남 대체 생산기지 부상

탐지 신호:
  - 특정 국가/지역 공급 비중이 50%+ 인 품목
  - 지정학 긴장 고조 + 대체 투자 발표
  - "공급망 다변화", "프렌드쇼어링" 보도 증가

전형적 타임라인:
  단기 (0~6개월): 공급 차단 충격, 가격 급등
  중기 (1~3년): 대체 경로 투자, 전환 비용 발생
  장기 (3~7년): 새 공급 구조 정착, 원래 공급자 지위 영구 약화
```

### 8. 위기 → 규제강화 → 진입장벽 (Crisis → Regulation → Barrier to Entry)

```
패턴명: 위기-규제-기존업체 수혜 사이클
메커니즘:
  산업/금융 위기 발생 → 정치적 압력 → 규제 강화 →
  규제 준수 비용이 진입장벽으로 작용 →
  기존 대형 업체(준수 여력 있음)가 수혜 →
  산업 집중도 심화

역사적 사례:
  - 2008 금융위기 → 도드-프랭크법 → 소형 은행 준수 비용 부담 → 대형 은행 시장 집중
  - FTX 사태 → 크립토 규제 강화 → 규제 준수 가능한 대형 거래소만 생존
  - 환경 사고 → 환경 규제 강화 → 대기업만 투자 가능한 설비 요구

탐지 신호:
  - "규제 강화" + "업계 부담" 보도
  - 소형 업체의 비용 증가 / 퇴출 뉴스
  - 대형 업체의 M&A 증가 (규제 활용한 시장 정리)

전형적 타임라인:
  단기 (0~1년): 위기 발생, 규제 논의 시작
  중기 (1~3년): 규제 시행, 준수 비용 발생
  장기 (3~5년): 산업 구조 재편, 집중도 상승
```

### 9. 성공 → 과잉확신 → 버블 (Success → Overconfidence → Bubble)

```
패턴명: 성공-과신-버블 사이클
메커니즘:
  초기 성공 → 성공 내러티브 확산 →
  레버리지 확대 + 신규 진입 가속 →
  자산 가격이 펀더멘탈 이탈 → 트리거 이벤트 → 급락

역사적 사례:
  - 닷컴: 인터넷 성공 → "이번에는 다르다" → NASDAQ 5000 → 붕괴
  - 2021 크립토: DeFi 성공 → "새 금융 패러다임" → BTC 69K → 15K
  - 2021 SPAC: 초기 SPAC 성공 → 남발 → 대부분 NAV 이하

탐지 신호:
  - "이번에는 다르다" 내러티브 확산
  - 레버리지 비율 역사적 고점 접근
  - 신규 계좌 개설 급증 / 비전문가 유입
  - 가격-수익 괴리 확대

전형적 타임라인:
  초기 (1~2년): 펀더멘탈 기반 상승, 합리적 낙관
  과열 (6개월~1년): 내러티브 지배, 레버리지 확대
  붕괴: 트리거 이벤트 → 급락 (시점 예측 불가, 조건만 식별 가능)
```

### 10. 탈세계화 → 비용상승 → 자동화 가속 (Deglobalization → Cost Up → Automation)

```
패턴명: 탈세계화-비용-자동화 사이클
메커니즘:
  리쇼어링/프렌드쇼어링 → 인건비 높은 국가에서 생산 →
  비용 상승 → 자동화/로봇 투자 가속 →
  제조업 일자리 증가 기대와 달리 자동화로 대체

역사적 사례:
  - 미국 리쇼어링 → TSMC 애리조나 팹 → 자동화율 극대화 (인력 최소화)
  - 유럽 탈중국 → 유럽 내 생산 시도 → 인건비 문제 → 로봇 도입 가속
  - (진행 중) 인도 제조업 이전 → 초기 인건비 장점 → 장기적 자동화 전환

탐지 신호:
  - 리쇼어링 발표 + 산업용 로봇 수주 증가
  - 신규 공장의 직원 수가 기존 대비 현저히 적음
  - "스마트 팩토리", "라이츠아웃 공장" 보도 증가

전형적 타임라인:
  단기 (0~2년): 리쇼어링 투자 발표, 건설 일자리 증가
  중기 (2~4년): 공장 가동, 예상보다 적은 고용
  장기 (4~7년): 자동화 완성, 제조업 고용 구조 변화
```

### 11. 인구구조 → 자산가격 → 정치경제 재편 (Demographics → Asset Repricing)

```
패턴명: 인구-자산-정치 사이클
메커니즘:
  고령화/인구 감소 → 자산 수요 구조 변화 →
  부동산/주식 수요 장기 하락 압력 →
  세대 간 갈등 → 정치적 재편 (복지 vs 성장)

탐지 신호:
  - 생산가능인구 감소 가속
  - 은퇴 세대 자산 처분 시작
  - "세대 갈등", "연금 개혁" 보도 증가

전형적 타임라인: 10~20년 단위의 구조적 흐름 (분석 시 장기 맥락으로 참조)
```

### 12. 데이터 독점 → 네트워크 효과 → 규제 불가능 (Data Moat → Regulatory Paradox)

```
패턴명: 데이터-네트워크-규제역설
메커니즘:
  데이터 축적 → 서비스 품질 향상 → 사용자 증가 →
  더 많은 데이터 축적 (자기강화 루프) →
  독점에 가까운 지위 → 규제하면 서비스 품질 하락 →
  규제의 역설 (규제하면 소비자 피해)

역사적 사례:
  - 구글 검색: 데이터 → 정확도 → 시장 지배 → 반독점 소송의 딜레마
  - 아마존: 판매 데이터 → 물류 최적화 → 가격 경쟁력 → 분할 어려움
  - OpenAI/ChatGPT: 사용자 데이터 → 모델 개선 → 사용자 증가 (진행 중)

탐지 신호:
  - 플랫폼 기업의 데이터 우위가 경쟁자 진입을 차단
  - 규제 논의에서 "소비자 후생" 논쟁 등장
  - 자기강화 루프가 가속 중

전형적 타임라인: 플랫폼 성숙 후 3~5년 시점에서 규제 논의 본격화
```

## 패턴 적용 원칙

```
1. 강제 적용 금지: 패턴이 현재 이슈에 맞지 않으면 적용하지 않는다.
   패턴은 사고의 촉매이지 답이 아니다.

2. 복수 패턴 허용: 하나의 이슈에 여러 패턴이 동시에 작용할 수 있다.
   예: 규제→회피→혁신 + 위기→규제→진입장벽 동시 작용

3. 타임라인 조정: 패턴의 전형적 타임라인은 참고값일 뿐.
   현재 이슈의 속도, 규모, 맥락에 맞게 조정한다.

4. 탐지 신호 활용: 패턴의 탐지 신호를 L7 추적 지표로 활용할 수 있다.
   "이 패턴이 진행 중인지 확인하려면 X를 추적하라."

5. 역사적 사례 참고: 과거 사례가 있다고 같은 결과가 보장되지 않는다.
   "이번에 다른 조건은 무엇인가?"를 항상 확인한다.
```
