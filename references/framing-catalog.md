# 언론 프레이밍 기법 카탈로그 (Framing Catalog)

> L1 헤드라인 디코딩에서 참조하는 프레이밍 패턴 라이브러리.
> 기사의 프레이밍을 먼저 식별해야 분석이 프레이밍에 오염되지 않는다.

---

## 사용법

1. L1에서 기사의 헤드라인과 리드를 읽는다
2. 아래 12개 패턴 중 해당하는 것을 식별한다 (복수 가능)
3. 프레이밍 태그를 부여한다: `[공포]` `[낙관]` `[권위호소]` 등
4. L4 이후 분석에서는 프레이밍을 벗기고 팩트만 놓고 진행한다

---

## 1. [공포] Fear Framing — 위협 강조

```
정의:
  부정적 결과를 과장하여 독자의 불안을 자극하는 기법.
  "~하면 큰일", "~의 공포", "~쇼크" 등의 표현.

탐지 방법:
  - 헤드라인에 "쇼크", "위기", "공포", "폭락", "붕괴", "최악" 등 부정 극단어
  - 본문 초반에 최악 시나리오를 먼저 제시
  - 수치를 절대값으로만 제시 (비율/맥락 없음)

예시:
  "AI 시대, 300만 일자리 사라진다" — 절대 수치만 제시, 새로 생기는 일자리 미언급
  "금리 폭등 쇼크... 부동산 시장 얼어붙나" — 극단 시나리오를 기정사실화

방어 전략:
  "이 위협이 현실화되려면 어떤 조건이 동시에 성립해야 하는가?"
  → 조건을 나열하면 공포의 크기가 객관화된다.
  반드시 L2에서 빠진 맥락(비율, 시간 프레임)을 보충한다.
```

## 2. [낙관] Optimism Framing — 긍정 과장

```
정의:
  긍정적 측면만 부각하고 리스크를 축소/생략하는 기법.
  "~의 기회", "~시대 개막", "사상 최대" 등의 표현.

탐지 방법:
  - 헤드라인에 "사상 최대", "돌파", "대박", "기회", "혁명" 등 긍정 극단어
  - 리스크/부작용이 본문 후반에 짧게 언급되거나 생략
  - 전문가 인용이 긍정 전망 일색

예시:
  "반도체 수출 사상 최대... K-칩 시대 열린다" — 편중 동향만 부각
  "비트코인 10만 돌파, 전문가들 '더 간다'" — 반대 의견 생략

방어 전략:
  "이 낙관이 성립하지 않는 조건은 무엇인가?"
  → L2에서 기사가 축소한 리스크를 "말하지 않은 것"으로 포착.
  → SCP 판독 시 낙관 프레이밍에 끌리지 않도록 주의 (R-02).
```

## 3. [권위호소] Authority Appeal — 전문가/기관 인용으로 신뢰 부여

```
정의:
  전문가, 기관, 보고서를 인용하여 주장의 신뢰도를 높이는 기법.
  인용 자체는 문제가 아니나, 선별적 인용이 문제.

탐지 방법:
  - "~에 따르면", "전문가 분석", "~연구소 보고서" 등 반복
  - 인용된 전문가의 이해관계 미공개
  - 반대 견해의 전문가 인용 부재
  - 기관의 보고서 중 일부만 발췌

예시:
  "골드만삭스 '한국 시장 비중 확대' 추천" — 골드만의 포지션/이해관계 미언급
  "교수 '이 정책은 실패할 것'" — 해당 교수의 정치적 성향/소속 미공개

방어 전략:
  "이 전문가/기관의 이해관계는 무엇인가?"
  → L3에서 인용 전문가를 플레이어 맵에 포함.
  → "인용되지 않은 반대 의견의 전문가는 누구인가?" 탐색.
```

## 4. [대비효과] Contrast Effect — 극단적 비교

```
정의:
  극단적인 두 대상을 비교하여 차이를 과장하는 기법.
  기준점을 의도적으로 선택하여 변화폭을 왜곡.

탐지 방법:
  - "A는 X인데 B는 Y" 형태의 극단 비교
  - 비교 기준점(baseline)이 의도적으로 선택됨 (최저점/최고점)
  - 맥락 없는 전후 비교 ("1년 전 대비 200% 상승" — 그 1년 전이 저점)

예시:
  "중국 GDP 성장 5%인데 한국은 겨우 2%" — 경제 규모/구조 차이 무시
  "주가 고점 대비 30% 하락" — 고점이 버블이었을 가능성 미언급

방어 전략:
  "비교 기준점은 왜 이 시점/대상인가?"
  → L2에서 기준점의 적절성을 검증.
  → 다른 기준점으로 비교하면 결론이 달라지는지 확인.
```

## 5. [모호] Ambiguity — 의도적 애매함

```
정의:
  핵심 정보를 애매하게 처리하여 독자가 원하는 대로 해석하게 하는 기법.
  "~할 수도", "~가능성", "~검토 중" 등 확정을 피하면서 인상은 남김.

탐지 방법:
  - "~할 수도 있다", "~가능성 배제 못 해", "~전망" 등 유보 표현 다수
  - 핵심 판단 주체가 불명확 ("관계자에 따르면")
  - 제목은 확정적이나 본문은 유보적 (제목-본문 괴리)

예시:
  "금리 동결 가능성... 시장 안도?" — 가능성일 뿐인데 시장 반응을 기정사실화
  "정부, 규제 완화 검토" — 검토 단계를 시행인 것처럼 보도

방어 전략:
  "확정된 것과 미확정인 것을 분리하면 무엇이 남는가?"
  → L2에서 확정 팩트와 추측/전망을 명확히 분리.
  → 제목과 본문의 괴리를 탐지하면 프레이밍으로 기록.
```

## 6. [긴급] Urgency Framing — 시간 압박

```
정의:
  시간적 긴박감을 조성하여 즉각적 반응(클릭, 행동)을 유도하는 기법.
  "속보", "긴급", "마지막 기회", "데드라인" 등.

탐지 방법:
  - "속보", "긴급", "지금 당장", "마감 임박" 등 시간 압박 표현
  - 실제 데드라인이 없는데 긴급성 부여
  - "지금 안 하면 늦는다" 형태의 FOMO 유발

예시:
  "속보: 미중 관세 전쟁 격화" — 수개월 진행 중인 상황을 '속보'로 포장
  "마지막 저가 매수 기회인가" — 근거 없는 시간 압박

방어 전략:
  "이것이 정말 24시간 내 판단이 필요한 이슈인가?"
  → Pre-Read Urgency와 기사의 긴급성 프레이밍을 분리.
  → 기사는 URGENT라고 하지만 실제는 WATCH/SLOW일 수 있다.
```

## 7. [숫자마법] Number Magic — 맥락 없는 수치

```
정의:
  숫자를 맥락(모수, 비율, 시간 프레임) 없이 제시하여 인상을 조작하는 기법.
  절대값/변화율을 선택적으로 사용.

탐지 방법:
  - 큰 절대값만 제시 (모수 대비 비율 없음)
  - 변화율만 제시 (절대값 없음, 기저효과 미고려)
  - 시간 프레임이 자의적 ("올해 들어" vs "최근 3개월" 등 편리한 구간 선택)

예시:
  "외국인 1조원 순매도" — 시총 대비 비율? 평소 대비 수준?
  "전년 대비 300% 성장" — 전년이 코로나 저점이었을 가능성

방어 전략:
  "이 숫자의 분모는 무엇인가? 기저효과는?"
  → L2에서 수치의 맥락(비율, 기저, 시간 프레임)을 반드시 보충.
  → [규모 렌즈] 돌발 질문으로 활용 가능.
```

## 8. [생존자편향] Survivorship Bias

```
정의:
  성공 사례만 보도하고 실패 사례를 누락하여 성공 확률을 과장하는 기법.
  "~로 대박", "~의 성공 비결" 등.

탐지 방법:
  - 성공 사례 N개 나열, 실패 사례 0개
  - "이렇게 하면 된다" 형태의 처방전 제시
  - 같은 전략을 쓰고 실패한 사례 미언급

예시:
  "비트코인 초기 투자자 100억 수익" — 같은 시기 투자하고 손실 본 사람 미언급
  "이 스타트업은 이렇게 유니콘이 됐다" — 같은 분야 100개 실패 기업 미언급

방어 전략:
  "이 성공 사례와 같은 조건에서 실패한 사례는 몇 개인가?"
  → L3에서 기사에 등장하지 않은 실패 사례를 탐색.
  → L6에서 base rate(기본 확률)를 고려.
```

## 9. [앵커링] Anchoring — 첫 숫자에 고정

```
정의:
  기사 초반에 특정 숫자/기준점을 제시하여 이후 판단을 그 숫자에 고정시키는 기법.

탐지 방법:
  - 기사 첫 문장에 인상적인 숫자 배치
  - 이후 모든 비교가 그 숫자 기준으로 진행
  - 다른 기준점(평균, 중간값, 추세)은 제시하지 않음

예시:
  "삼성전자 사상 최고가 10만원 대비..." — 10만원이 앵커가 되어 현재가를 저평가로 인식
  "목표가 15만원... 현재 7만원" — 목표가가 앵커가 되어 상승 여지를 과대 인식

방어 전략:
  "이 숫자 대신 다른 기준(52주 평균, 업종 평균)으로 보면?"
  → L2에서 앵커 숫자를 식별하고 대안 기준점을 제시.
  → SCP 판독 시 앵커에 끌리지 않도록 주의.
```

## 10. [프레임전환] Frame Shifting — 논점 이탈

```
정의:
  핵심 논점에서 다른 논점으로 은근히 전환하여 본질을 회피하는 기법.
  A 문제를 다루다가 슬쩍 B 문제로 넘어감.

탐지 방법:
  - 제목의 논점과 본문의 결론이 다른 주제
  - "이와 관련하여"로 시작하는 논점 전환
  - 핵심 질문에 대한 직접 답변 회피

예시:
  제목: "반도체 수출 감소 심각" → 본문 결론: "정부 지원 확대 필요"
  (수출 감소의 원인 분석 → 정부 책임으로 전환)

방어 전략:
  "기사의 원래 질문은 무엇이었고, 실제로 답했는가?"
  → L1에서 제목의 논점과 본문의 결론을 대조.
  → L4에서 "왜 논점이 전환되었는가"를 추적.
```

## 11. [감정이입] Emotional Appeal — 개인 사연으로 일반화

```
정의:
  개인의 사연(감정적 스토리)을 제시한 뒤 이를 전체 현상으로 일반화하는 기법.
  통계적 근거 없이 감정적 공감으로 주장을 뒷받침.

탐지 방법:
  - "A씨(가명)는..." 으로 시작하는 개인 사연
  - 개인 사연 후 "이처럼 많은 사람들이" 식의 일반화
  - 통계/데이터 없이 사연 하나로 추세 주장

예시:
  "30대 직장인 A씨는 영끌 후 이자 폭탄..." → "이처럼 많은 2030이 위기"
  (A씨 사례 = 개인, "많은 2030" = 검증 안 된 일반화)

방어 전략:
  "이 사연이 대표적인가? 전체 중 몇 %에 해당하는가?"
  → L2에서 개인 사연과 전체 통계를 분리.
  → 감정 분리 원칙 적용: 사연에 공감하되 분석은 팩트로.
```

## 12. [선택적생략] Selective Omission — 불리한 사실 제거

```
정의:
  기사의 주장에 불리한 사실이나 맥락을 의도적으로 생략하는 기법.
  거짓말은 아니지만 진실의 일부만 전달.

탐지 방법:
  - 한쪽 데이터만 제시 (상승만 or 하락만)
  - 시간 범위가 편향적 (유리한 구간만 선택)
  - 관련 핵심 사실이 빠져 있음 (다른 기사에서는 언급)

예시:
  "수출 3개월 연속 증가" — 같은 기간 수입 증가로 무역수지 악화는 생략
  "A기업 매출 50% 성장" — 영업이익 적자 전환은 생략

방어 전략:
  "이 기사가 의도적으로 빠뜨렸을 사실은?"
  → L2의 "기사가 말하지 않은 것"이 이 패턴을 직접 방어.
  → Tavily 추가 수집으로 다른 기사의 관점을 교차 확인.
```

## 복합 프레이밍

하나의 기사에 여러 프레이밍이 동시에 적용될 수 있다.

```
흔한 조합:
  [공포] + [긴급] + [숫자마법] — "폭락 쇼크, 1조 증발, 지금이 마지막"
  [낙관] + [권위호소] + [앵커링] — "골드만 '목표가 15만원', 역대 최고 실적"
  [감정이입] + [선택적생략] — 개인 사연으로 감정 유발 + 반대 사례 생략
  [대비효과] + [프레임전환] — 극단 비교 후 논점을 다른 곳으로 이동

복합 프레이밍은 개별 태그를 모두 부여한다:
  프레이밍: [공포][긴급][숫자마법]
```
