# 자율적 사고 가이드 (Autonomous Thinking Guide)

SKILL.md의 Phase 0(Pre-Read), Phase 2.5(되먹임), Phase 3(돌발 질문)의
상세 판단 기준과 실전 예시를 담고 있다.

---

## 1. Pre-Read 판독의 실전 판단 기준

### Type 분류가 애매할 때의 결정 규칙

여러 Type이 겹칠 때 **가장 구조적 영향이 큰 것**을 주(主) Type으로 선택한다.

```
판단 우선순위:
STRUCT > POLICY > MACRO > NARR > EVENT > NOISE

예시:
"미국, AI반도체 수출규제 강화" 
→ POLICY인가 STRUCT인가?
→ 규제가 밸류체인 재편을 유발하면 POLICY×STRUCT (주: STRUCT)
→ 규제가 일시적 무역마찰이면 POLICY (주: POLICY)
→ 판단 근거: "이 규제로 인해 공급망 재배치가 발생하는가?"

"테슬라 로보택시 공개"
→ EVENT인가 STRUCT인가?
→ 기존 이동수단 밸류체인을 바꿀 수 있으면 EVENT×STRUCT (주: STRUCT)
→ 마케팅 이벤트에 그치면 EVENT
→ 판단 근거: "이 발표 후 다른 플레이어들의 전략이 바뀌는가?"
```

### SCP 판독의 함정 방지

SCP를 과대/과소 판독하는 흔한 오류:

```
과대 판독 경향:
├── "최초", "사상 최대", "전례 없는" 같은 수식어에 끌려감
│   → 방어: 수식어를 제거하고 팩트만 놓고 SCP를 판독한다
├── 한 기업의 이벤트를 산업 구조 변화로 확대 해석
│   → 방어: "이것이 다른 기업에도 적용되는가?"를 묻는다
└── 기사의 프레이밍(공포/탐욕)이 SCP 판독에 오염됨
    → 방어: L1 프레이밍 태그를 먼저 식별한 후 SCP 판독

과소 판독 경향:
├── 작은 숫자/소규모 사건이라 무시 → 초기 신호 놓침
│   → 방어: "이것이 3년 뒤에도 작을까?"를 묻는다
├── 익숙한 이슈라 반복으로 분류 → 실은 전환점
│   → 방어: "이전과 정확히 뭐가 다른가?"를 묻는다
└── 단발 이벤트로 보여서 구조와 무관하다고 판단
    → 방어: "이 이벤트가 연쇄 반응을 일으킬 수 있는가?"를 묻는다
```

### 감정 분리 실전 예시

```
예시 1: "삼성 실적 또 빠졌네... 이제 끝난 거 아냐?"
  Step 1: "이 걱정의 핵심은 삼성의 경쟁력 하락이 비가역적인지에 대한 불확실성"
  Step 2: "삼성전자의 반도체 사업부 경쟁력이 비가역적으로 훼손되는 조건은?"
  Step 3: 분석 진행...
  L7 회답: "비가역적 훼손 조건: ①HBM4 양산 실패 ②파운드리 고객 이탈 가속
           현재: ①진행 중이나 확정 아님 ②일부 이탈이나 삼성 고유 수요 존재
           → 🟡 시그널 수준이지만 즉시 '끝'은 아님. KC: HBM4 수율 데이터(Q3)"

예시 2: "비트코인 10만 돌파!! 대박 더 갈 수 있겠지??"
  Step 1: "이 흥분의 핵심은 추가 상승 모멘텀의 지속 여부에 대한 기대"
  Step 2: "BTC가 10만 이후 추가 상승하기 위한 구조적 조건과 소진 조건은?"
  Step 3: 분석 진행...
  L7 회답: "추가 상승 조건: ①ETF 순유입 유지 ②글로벌 유동성 확대 지속
           소진 조건: ①ETF 유입 둔화→유출 전환 ②레버리지 청산 연쇄
           → 현재 ①유지 중이나 속도 감소, ②레버리지 비율 과거 고점 근접
           → 🟡 모멘텀은 있으나 과열 구간. KC: ETF 일일 순유출 3일 연속"
```

## 2. 되먹임(Feedback Loop) 상세 판단 기준

### 어떤 FB를 실행할지 선택하는 규칙

전부 기계적으로 돌리면 형식적이 된다.
Pre-Read 결과에 따라 가장 가치 있는 1~2개만 선택한다.

```
Type별 우선 FB:

POLICY  → FB-1 (L4→L2) 필수, FB-2 (L5→L3) 권장
  이유: 정책 의도를 알면 기사가 숨긴 팩트가 드러남.
        규제 구조를 알면 숨은 이해관계자가 보임.

MACRO   → FB-1 (L4→L2) 필수, FB-3 (L6→L7) 권장
  이유: 매크로 원인을 알면 기사가 과장/축소한 팩트가 보임.
        2차 효과가 크면 KC가 바뀜.

STRUCT  → FB-2 (L5→L3) 필수, FB-3 (L6→L7) 권장
  이유: 구조 변화의 수혜/피해자가 표면에 안 나옴.
        밸류체인 재편의 2차 효과가 판단을 바꿈.

EVENT   → FB-4 (L7→L1) 만으로 충분한 경우 많음
  이유: 단발 이벤트는 구조를 크게 바꾸지 않으므로
        최종 대조만으로 분석 품질 점검 가능.

NARR    → FB-1 (L4→L2) 필수
  이유: 내러티브 전환의 원인을 알면,
        "프레이밍이 팩트를 왜곡하는 방식"이 구체적으로 보임.

NOISE   → FB 불필요 (L1에서 종료하므로)
```

### FB 실행 후 "보강할 것이 있는가"를 판단하는 기준

되먹임을 실행했다고 반드시 수정하는 것이 아니다.
아래 조건 중 하나라도 해당하면 보강한다:

```
보강 조건:
├── FB-1: L4 분석 결과, L2에 기록하지 않았지만 존재했어야 할 
│         팩트가 구체적으로 식별되는가? (예: 기사가 의도적으로 뺀 비교 데이터)
│
├── FB-2: L5 구조 분석 결과, L3에 없는 플레이어가 실제로 
│         이 이슈에서 중요한 역할을 하는가? (예: 기사에 안 나온 규제 기관)
│
├── FB-3: L6의 2차 효과 크기가 L7에서 설정한 KC의 
│         임계값을 바꿀 정도인가? (예: 2차 효과가 원래 이슈보다 큼)
│
└── FB-4: L1을 다시 읽었을 때, 같은 문장이 다른 의미로 
          읽히는가? (예: "위기"라는 단어가 이제 "기회"로 보임)
```

## 3. 돌발 질문(Emergent Questions) 생성의 실전 가이드

### 렌즈 선택 기준

```
Type별 우선 렌즈:

POLICY  → [반전 렌즈] + [침묵 렌즈]
  "이 규제가 실패하면 어떤 증거가 먼저 나오는가?"
  "이 규제에 대해 가장 조용한 기업은? 왜?"

MACRO   → [규모 렌즈] + [역사 렌즈]
  "이 숫자가 전체 GDP/시장의 몇 %인가?"
  "과거 유사 금리 사이클에서 시장은 어떻게 반응했는가?"

STRUCT  → [시간 렌즈] + [침묵 렌즈]
  "이 구조 변화가 3년 전에 시작됐다면 지금 어디쯤인가?"
  "이 밸류체인에서 아직 움직이지 않은 플레이어는?"

EVENT   → [반전 렌즈] + [규모 렌즈]
  "이 실적이 기대치를 충족했다면 주가 반응이 달랐을까?"
  "이 M&A 금액은 인수자 시총의 몇 %인가?"

NARR    → [역사 렌즈] + [시간 렌즈]
  "과거에 이 내러티브가 지배했을 때, 실제 결과는?"
  "이 심리 전환이 6개월 일찍 왔다면 시장 위치가 달랐을까?"
```

### 좋은 돌발 질문의 조건

```
좋은 질문: 
├── 답을 모르지만, 답을 알면 판단이 극적으로 달라진다
├── 기사의 프레이밍 안에서는 나올 수 없는 질문이다
├── 검증 가능하다 (어떤 데이터/이벤트로 답할 수 있는지 특정 가능)
└── 다른 사람들이 아직 묻지 않고 있다

나쁜 질문 (피해야 할 것):
├── 너무 일반적 ("앞으로 어떻게 될까?")
├── 이미 기사가 답한 것 ("실적이 좋았나 나빴나?")
├── 검증 불가능 ("시장 참여자들의 심리 상태는?")
└── 답이 뻔한 것 ("물가가 오르면 금리를 올리나?")
```

## 4. 불확실성 지도 작성 기준

### Layer별 확신도 판정 근거

```
■■■■■ 높음 조건:
  - 복수의 독립 출처에서 확인된 팩트
  - 공식 발표/데이터에 기반
  - 과거 전례가 명확하고 현재 조건이 전례와 유사

■■■□□ 중간 조건:
  - 단일 출처이나 신뢰할 만한 기관/언론
  - 합리적 추론이나, 반대 논거도 존재
  - 전례가 있으나 현재 조건이 부분적으로 다름

■□□□□ 낮음 조건:
  - 출처 없이 추론만으로 도출
  - "~일 수 있다", "~할 가능성이 있다"로만 기술 가능
  - 전례가 없거나, 전례와 현재의 유사성이 약함
  - 2차, 3차 효과 연쇄 추론 (연쇄가 길수록 확신도 하락)
```

### Layer 유형별 기본 확신도 범위

L1~L2는 팩트 기반이라 확신도가 높은 편이고,
L5~L6은 구조 판단·미래 추론이라 확신도가 낮은 편이다.
이 기본값에서 구체적 근거에 따라 조정한다.

```
L1 (헤드라인): 기본 ■■■■■ — 기사 텍스트 자체이므로 거의 항상 높음
L2 (팩트):     기본 ■■■■□ — 출처 확인 여부에 따라 변동
L3 (이해관계): 기본 ■■■□□ — 공개 정보 기반이나 '숨은 동기'는 추론
L4 (인과):     기본 ■■□□□ — 구조적 원인은 항상 가설적 요소 포함
L5 (구조):     기본 ■■■□□ — 전례 유무에 따라 큰 폭으로 변동
L6 (2차효과):  기본 ■□□□□ — 미래 추론이므로 기본적으로 낮음
L7 (판단):     기본 ■■■□□ — 조건부 형태이므로 중간, KC의 구체성에 따라 변동
```
