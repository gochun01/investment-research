# investment-research

투자 리서치 자료 저장소입니다.

## 구조

```
investment-research/
├── README.md
├── reports/
│   ├── crypto/          ← LINK, ETH, BTC 등 암호화폐 리포트
│   └── equity/          ← 삼성전자, NVDA 등 주식 리포트
└── skills/
    ├── SKILL.md         ← 분석 스킬 개요
    ├── SECTIONS.md      ← 리포트 섹션 구성
    ├── EXAMPLES.md      ← 분석 예시
    └── GUIDE.md         ← 작성 가이드
```

## 디렉토리 설명

### reports/
투자 분석 리포트를 자산군별로 구분하여 보관합니다.
- `crypto/` : 암호화폐 (BTC, ETH, LINK 등) 분석 리포트
- `equity/` : 주식 (삼성전자, NVDA 등) 분석 리포트

### skills/
리서치 및 분석에 활용되는 방법론과 가이드라인을 담습니다.
- `SKILL.md` : 핵심 분석 스킬 정리
- `SECTIONS.md` : 리포트 섹션별 구성 방법
- `EXAMPLES.md` : 실제 분석 예시
- `GUIDE.md` : 리서치 작성 전반 가이드
