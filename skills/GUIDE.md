# GUIDE

## 리서치 작성 전반 가이드

투자 리포트 작성부터 관리까지의 전체 프로세스를 안내합니다.

---

## 1. 리서치 시작 전 쭐크리스트

- [ ] 자산 유형 확인 (Crypto / Equity)
- [ ] 파일명 규칙 설정 (`YYYYMMDD_TICKER.md`)
- [ ] 저장 디렉토리 확인 (`reports/crypto/` 또는 `reports/equity/`)
- [ ] 선행 리포트 마지막 업데이트일 확인

## 2. 파일명 규칙

```
YYYYMMDD_TICKER.md

예시:
20260222_LINK.md
20260222_NVDA.md
20260222_005930.md
```

## 3. 리포트 작성 프로세스

### Step 1: 정보 수집
- 관련 뉴스 및 공식 렀포트 수집
- 온체인 / 재무 데이터 확인
- 거시경제 맥락 정리

### Step 2: 스케치 작성
- SECTIONS.md 가이드 참조
- 핸드라이팅으로 코어 녹리 정리

### Step 3: 분석 작성
- SKILL.md의 분석 도구 적용
- EXAMPLES.md 참조하여 형식 일관성 유지

### Step 4: 업데이트 루틴
- 주미 1회 이상 리뷻 권장
- 주요 이벤트 발생 시 즉시 업데이트

## 4. 품질 기준

- 폴치 교체는 데이터 근거로만
- 주관적 표현 지양, 수치/지표로 뒷받침
- 다양한 시나리오 고려 (Bull / Base / Bear)
- 리스크 요인 나열 필수

## 5. 형상 관리

- reports/ 폴더는 자산군별로 어엄
- 아카이브하지 않고 모든 리포트 유지 (리비져 이력 활용)
- 쿤더리 📄 없이 파일명으로 연대순 파악
